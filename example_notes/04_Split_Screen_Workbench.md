# Split Screen Workbench

Use this note to test split panes.

## Steps

1. Open [[00_Test_Index]] as the main note.
2. Right-click this note in Files and choose "Open in Split".
3. Confirm a split pane opens below the main note.
4. Edit this line in the split pane: `split edit smoke test`.
5. Confirm the title shows an unsaved marker.
6. Click the save icon.
7. Click the vertical swap icon and confirm this note becomes the main note while the previous main note appears in this split pane.

## Live rendering in split pane

Typing **bold**, *italic*, `code`, ==highlight==, #split-tag, and headings should update live in this split pane.

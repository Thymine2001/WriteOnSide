# WriteOnSide Example Notes Test Vault

Set WriteOnSide's notes folder to this `example_notes` directory, then open `00_Test_Index.md`.

This test Vault covers:

- Markdown live highlighting and read-mode rendering
- Frontmatter metadata, aliases, and tag filtering
- Wikilinks, heading links, block links, note embeds, backlinks, and rename updates
- Outline, sticky headings, find/replace, and code copy
- File explorer search, context actions, nested folders, text-file editing, and previews
- Right-click split opening, drag/drop attachments, image links, image preview, and split panes
- Settings for theme, typography, layout, opacity, language, and shortcuts

The notes reference `assets/icon_light.png`, so image preview tests are self-contained when `example_notes` is selected as the notes folder.

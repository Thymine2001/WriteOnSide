# Files, Attachments, and Previews

Use this note to test file explorer actions and attachment workflows.

## File explorer actions

- Create a new note from the toolbar and from Files.
- Rename a note.
- Copy, cut, paste, and delete a disposable note.
- Right-click files and use Copy Path, Reveal, Preview, Edit, and Open Externally where available.
- Expand and collapse nested folders under `Projects`.

## Drag/drop tests

- Right-click another Markdown note in Files and choose "Open in Split".
- Drag an image file into this note to copy it into the configured attachments folder.
- Paste a clipboard image into this note.
- Drag a folder into Files to copy it into the Vault.

# Settings, Themes, and Shortcuts

Use this note while changing settings.

## Appearance

- Change theme and verify the main editor, split panes, Files, tag panel, nav strip, and tooltips update.
- Change font family and size. Main note and split panes should both update.
- Change opacity and panel width.
- Change Files width and verify file names still fit.

## Layout

- Switch left/right panel position.
- Resize the panel from the edge.
- Resize the Files panel.
- Toggle Files open/closed.

## Shortcuts

- Verify global panel hotkey.
- Change toolbar shortcuts.
- Use `Ctrl+B`, `Ctrl+I`, find, replace, outline, backlinks, and save commands.

## Language

Switch between English, Simplified Chinese, and Portuguese.

Verify:

- Toolbar tooltips update.
- Split pane save tooltip updates.
- Split pane swap tooltip updates.
- Settings labels update.
